JavaScript Brick Breaker


About this game

Controls: Left and right arrow keys move the paddle. Space retries if all lives are exhausted.

If you're on a mobile device, it's pretty obvious that I didn't design this game for mobile use. This project was a short refresher on JavaScript and an interesting look into the way people make games using JavaScript as this was something I had never been exposed to. I started off by following this tutorial on how to make a 2D game with only JavaScript, but was annoyed with some of the major problems with the game's "physics" and decided to expand on it and make it a little bit more my own.

This version takes horizontal bounces into account, adds directional bounces based on where the ball hits the paddle, has a more clear-cut start and end, makes use of a more user-friendly respawn sequence in between lives, and adds a little bit more color. This was a brief, but fun project that I enjoyed creating. Hope you like it.
